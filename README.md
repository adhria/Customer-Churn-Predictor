# Customer Churn Predictor

## Overview
Telecom churn prediction using Logistic Regression and Random Forest.  
End-to-end pipeline: data cleaning → model training → evaluation → business insights.

## Dataset
Telco Customer Churn dataset (IBM, available on Kaggle).

## Models
- Logistic Regression (Accuracy ~82%, Recall ~60%)
- Random Forest (Accuracy ~78%, Recall ~44%)

## Insights
Top churn drivers: month-to-month contracts, lack of online security, shorter tenure.  
Recommendation: promote long-term contracts, bundle online security, target retention campaigns.
